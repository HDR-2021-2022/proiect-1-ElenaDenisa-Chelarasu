Limbajul HTML

HTML(Hyper Text Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod HTML il gasiti in imaginea urmatoare care se afla la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png . Imaginea are atributul height de valoare 127.

Tag-urile html cele mai utilizate in realizarea tabelelor sunt:

Comanda Functionalitate
table defineste un tabel
tr un rand dintr-un tabel
td o celula dintr-un tabel

Observatie: Pentru a pune un contur unui tabel se poate (dar nu se recomanda) sa se puna atributul border="1".
La validare va da eroarea: The boder attribute on the table element is obsolete.
